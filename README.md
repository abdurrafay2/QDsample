# Global Travel Club

A clean, minimalist travel booking website built with Next.js and Tailwind CSS.

## Features

- **Search-First Design**: Clean, professional interface focused on booking
- **Multi-Service Booking**: Hotels, Airlines, and Transport booking in one widget
- **Responsive Design**: Works seamlessly on all devices
- **Modern UI**: Clean typography, lots of white space, professional appearance

## Tech Stack

- Next.js 14 (App Router)
- React 18
- TypeScript
- Tailwind CSS
- Inter Font

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Project Structure

- `app/` - Next.js App Router pages and layouts
- `components/` - Reusable React components
- `app/globals.css` - Global styles and Tailwind imports

## Key Components

- **BookingWidget**: Main booking component with tabbed interface
- **Homepage**: Clean landing page with hero section and feature highlights
- **Navigation**: Minimal navigation bar with key links

## Design Philosophy

This website follows a minimalist, search-first approach inspired by modern booking engines like Google Flights. The design prioritizes:

- Clean, uncluttered interface
- Clear call-to-action buttons
- Professional typography
- Generous white space
- Trustworthy appearance
